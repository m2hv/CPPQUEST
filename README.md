# CppQuest

## Project Description
CppQuest is an educational web application designed to help beginners learn C++ programming through structured lessons, coding examples, quizzes, and practice exercises. The platform provides an interactive and user-friendly learning experience for students who want to build their C++ programming skills.

## Features
- Structured learning path
- Interactive quizzes
- Coding examples
- Practice exercises
- Progress tracking
- Responsive user interface

## Technologies Used
- Lovable
- HTML
- CSS
- JavaScript
- AI Assistance
